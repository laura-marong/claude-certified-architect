# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 仓库用途

Claude 认证架构师（Claude Certified Architect）考试备考笔记，覆盖全部 5 个考试领域。

## 领域文件说明

- `01-智能体架构与编排.md` — 智能体循环、多智能体编排（Hub-and-Spoke）、钩子、会话管理
- `02-工具设计与MCP集成.md` — 工具描述设计、结构化错误响应、工具分发、MCP 配置、内置工具
- `03-Claude-Code配置与工作流.md` — CLAUDE.md 层级、自定义命令与技能、计划模式、CI/CD 集成
- `04-上下文管理与可靠性.md` — 上下文优化、"案件事实"块、升级与错误传播、信息溯源
- `05-提示工程与结构化输出.md` — 显式标准、少样本提示、tool_use 结构化输出、验证-重试循环

## 学习路线（推荐阅读顺序）

按概念依赖排序，而非官方领域编号：

```
01-智能体架构与编排        (核心：Agent 循环、编排、钩子)
       ↓
02-工具设计与MCP集成       (能力扩展：工具设计、MCP 集成)
       ↓
03-Claude-Code配置与工作流 (落地：团队工作流、CI/CD 集成)
       ↓
04-上下文管理与可靠性      (可靠性：长期上下文、升级、溯源)
       ↓
05-提示工程与结构化输出   (基础：显式标准、少样本、结构化输出)
```

## 格式约定

- 所有文件使用简体中文撰写
- 使用表格对比"反模式 ❌ vs 正确做法 ✅"
- 每个领域末尾包含考试提示汇总和导航链接
