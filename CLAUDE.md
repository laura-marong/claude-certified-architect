# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 仓库用途

Claude 认证架构师（Claude Certified Architect）考试备考笔记，覆盖全部 5 个考试领域。

## 领域文件说明

- `agentic-architecture.md` — 智能体循环、多智能体编排（Hub-and-Spoke）、钩子、会话管理
- `tool-design-mcp.md` — 工具描述设计、结构化错误响应、工具分发、MCP 配置、内置工具
- `claude-code-config.md` — CLAUDE.md 层级、自定义命令与技能、计划模式、CI/CD 集成
- `prompt-engineering.md` — 显式标准、少样本提示、tool_use 结构化输出、验证-重试循环
- `context-management.md` — 上下文优化、"案件事实"块、升级与错误传播、信息溯源

## 格式约定

- 所有文件使用简体中文撰写
- 使用表格对比"反模式 ❌ vs 正确做法 ✅"
- 每个领域末尾包含考试提示汇总和导航链接
